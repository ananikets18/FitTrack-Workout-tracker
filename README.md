# FitTrack - Workout Tracker

Modern PWA workout tracker with offline support, data import/export, and progress tracking.

## Key Features

- Workout logging with exercise autocomplete (150+ exercises)
- Progressive overload tracking & statistics
- Data import/export (JSON/Excel)
- PWA with offline support
- Rest timer with presets

## Tech Stack

React 19, Vite 7, Tailwind CSS, Framer Motion, Context API

## Project Structure

```
src/
├── components/  # UI components
├── pages/       # Routes
├── context/     # State management
```
